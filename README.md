ZEGL
====

Game development with SDL and OpenGL

Windows Development
-------------------
- Visual Studio 2013
- SDL2 (https://www.libsdl.org/download-2.0.php)
- GLEW (http://glew.sourceforge.net/)

Mac OS X Development
--------------------
- Xcode 6.x
- SDL2 (https://www.libsdl.org/download-2.0.php)
- GLEW (brew install libGLEW)
