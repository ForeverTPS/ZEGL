Detailed Description

Using a namespace rather than a static class or singleton etc to provide a robust but lightweight logging system.

void Logger::Init ( const std::string &  fname = "ZEGL")

Initialize the log.

Created the file and logs the start time. The file will be created in the user home directory under a folder ZEGL with the passed file name and a html extension.
